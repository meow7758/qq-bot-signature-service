# 在 n8n 服务器上安装 Ed25519 支持

## 方法 A：SSH 到 NAS 安装（推荐）

1. **SSH 连接到您的 NAS**

2. **找到 n8n 安装目录**：
   ```bash
   # 如果是 Docker 部署
   docker exec -it n8n bash

   # 如果是 npm 安装
   cd /path/to/n8n
   ```

3. **安装 tweetnacl 库**：
   ```bash
   npm install tweetnacl tweetnacl-util
   ```

4. **重启 n8n**：
   ```bash
   # Docker
   docker restart n8n

   # npm
   pm2 restart n8n
   # 或
   systemctl restart n8n
   ```

## 方法 B：在 Docker Compose 中配置

如果使用 Docker Compose，修改 `docker-compose.yml`：

```yaml
services:
  n8n:
    image: n8nio/n8n
    command: npm install tweetnacl tweetnacl-util && n8n start
    # ... 其他配置
```

然后重启：
```bash
docker-compose down
docker-compose up -d
```
